# setup_basic_vm.sh

## Description
This script let you create a VM using VirtualBox SDK.

### How to use setup_basic_vm.sh
Execute as follow:
```
./setup_basic_vm.sh
```

# clone_vm.sh

## Description
Clone a vm from another that already exist.

### How to use clone_vm.sh
Execute as follow:
``` ./clone_vm.sh <NAME_FOR_NEW_VM>
```
